# Blog_os

這是參考使用 Rust 寫一個 OS 的[blog](https://os.phil-opp.com/)

## environment

- qemu
- rust

## How run

```
git clone https://github.com/onon1101/blog_os.git -b part02
cargo install bootimage
rustup component add llvm-tools-preview
cargo run
```
